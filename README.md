# Smart Food Ordering System

A responsive frontend web application for ordering food online.

## Features

- Login & Signup Page
- Dashboard
- Food Cards
- Search Bar
- Add to Cart
- Cart Summary
- Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
